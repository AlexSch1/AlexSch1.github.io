平台架构

任何交易的 API 密钥

任何提供API 支持的交易所将可在Membrana平台上操作。

智能ORACLES

Oracles 为智能合约提供了区块链外的数据。在Membrana平台上，所有交易数据都会通过Oracles。

以太坊(ETHEREUM)智能合约

交易者和投资者之间的所有交易都通过智能合约来进行。我们绝对不会把您的资金储存在我们的钱包里。

如何运作:

Membrana在投资者和交易者之间建立了直接的联系，使他们的关系能互惠互利, 并且为双方提供了担保。投资者的资金永远不会离开他们的交易钱包，所有的信托管理都会通过所获得的API来执行。每个参与者的关系都将通过智能合约，不但消除了中间人和交易对手的风险，还能强制达成协议。

我们在智能合约中记录了所有的交易细节：

  • 投资者账户中数字资产的初始数额 (比特币BTC、以太坊 ETH或美元 USD)
  • 完成合约的目标利润 (基于交易者的平均利润率。完成后, 合约将自动终止)
  • 合约期限（合同将自动计算利润并将其发送给交易者）
  • 最大损失（如果损失达到最大值，权限将自动关闭。合约也会解散，交易者费用将会被退还给投资者）
  • 授权的交易对
  • 交易者费用和他的钱包

技术

以太坊 (Ethereum)

Membrana以以太坊 (Ethereum) 为基础构建而成，是卓越的区块链协议，使先进的智能合约能自动运行，无需人工控制。以太坊是安全并去中心化的;它为平台上的所有操作提供了透明度。

Metamask

为了让我们的平台更安全，Membrana所有的流程都使用MetaMask完成。Metamask为一个Chrome插件，允许您在浏览器中正确运行以太坊去中心应用程式（Ethereum DApps），而无需运行完整的以太坊节点 (Ethereum node)。 MetaMask涵盖了一个安全的身份存储库，提供用户界面来管理和签署区块链交易。
