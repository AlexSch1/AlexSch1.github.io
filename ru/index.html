Компоненты платформы Membrana

Платформа для инвесторов

Инвестор может выбрать лучшего трейдера и заключить с ним контракт в 3 шага:

  • Добавить API ключ любой биржи
  • Отправить запрос выбранному трейдеру с заданными параметрами и выбранными парами для торговли
  • Отправить MBN токены на адрес смарт контракта (комиссия трейдера)
  • Подписать смарт контракт для начала торгов

Прибыль начисляется прямо на биржевой счет инвестора.

Платформа для трейдеров

Трейдер может получить деньги в доверительное управление сделав следующие шаги:

  • Начать торговать через платформу Membrana
  • Задать собственные условия для доверительного управления
  • Достигнуть топовых позиций в рейтинге трейдеров
  • Принять запрос от инвестора и подписать контракт с ним
  • Торговать, используя технологию доступа через API ключи

Прибыль трейдера гарантирована смарт контрактом. Он получит прибыль напрямую на собственный ETH кошелек.

Платформа для Фондов

Инвестор или трейдер могут быть частью или создать собственный Фонд в несколько шагов:

  • Создать торговый или инвестиционный фонд и приглашать новых участников
  • Настроить условия работы Фонда
  • Создать предложение для участия в голосовании
  • Корпоративные действия управляются через multisig

Фонд получает прибыль напрямую на свой счет на бирже.

Как устроена платформа

API ключи любой биржи

Любая биржа с поддержкой торговли через API ключи доступна для платформы Membrana.

Умные Oracles

Oracles поставляют данные, существующие вне блокчейна, для смарт контрактов. У платформы Membrana все данные поступают через Oracles.

Смарт контракты на Ethereum

Все сделки между трейдером и инвестором заключаются через Смарт контракт. Мы никогда не храним ваши деньги на наших кошельках.

Как это работает

Membrana устанавливает прямую связь между инвестором и трейдером, делая их взаимоотношения выгодными и надежными для обеих сторон. Средства инвесторов никогда не покидают кошелька на бирже, все доверительное управление выполняется благодаря API ключам. Любые взаимоотношения совершаются через смарт контракт, исключая посредников и контрагентов.

Все параметры сделки записываются в смарт контракт:

  • Начальные балансы цифровых активов на аккаунте инвестора (BTC, ETH или USD);
  • Целевая прибыль для завершение контракта (основывается на средней доходности трейдеров. При завершении контракта между трейдером, смарт контракт автоматически прекращается);
  • Длительность контракта (контракт автоматически вычисляет прибыль на протяжении всего периода и высылает трейдеру);
  • Максимальные потери (если достигается уровень максимальных потерь, доступ автоматически закрывается, контракт завершается и комиссия трейдера возвращается инвестору);
  • Доступные пары для торгов;
  • Комиссия трейдера и его кошелек.

Технологии

Ethereum

Membrana построена на Ethereum, блокчейн протоколе нового поколения, включающем продвинутые смарт контракты - соглашения, которые выполняются автоматически без контроля со стороны человека. Ethereum децентрализован и безопасен. Благодаря Ethereum все операции на платформе прозрачны и надёжны.

Metamask

Чтобы сделать нашу платформу более защищенной, все процессы выполняются через MetaMask. Это плагин для Chrome, который позволяет запускать Ethereum DApps прямо в Вашем браузере без запуска полной ноды Ethereum. MetaMask использует защищенный внутренний интерфейс, позволяющий пользователям управлять транзакциями в блокчейне.
