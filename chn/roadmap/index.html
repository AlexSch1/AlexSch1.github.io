Roadmap

Q1 2017
概念起源

确立加密资产的信托管理之理念。

找出信托管理市场中各大问题的解决方案，如：

  • 失去对参与资本的控制权
  • 来自中间人和交易对象的风险
  • 缺乏信任
  • 诈骗管理结果的风险
Read more
Q2 2017
绩效审计

将此概念应用于全绩效审计。

探索其在各种状况中的适应性，进行额外的安全审计以确保参与者的资本安全。
研究和分析所有潜在风险，调整或改变运作流程以消除风险。

Read more
Q3 2017
创建界面

创建平台架构并塑造功能齐全的用户界面。

组建专业的全职工作团队，拥有设计、UI / UX、前端/后端、可靠性语言，营销和安全方面的专家。
打造当前版本的设计。

Q3 2019
高级功能

分析器，投资组合经理，MBN基金

  • 安全合同。交易者在合约中保留自己的资金，以部分弥补投资者可能遭受的损失
  • 货币新闻递送。能同时在交易终端回顾特定货币的新闻和价格变化图表
  • 分析器—跟踪某些特定货币的数量，并通过通知系统提供信息
  • 投资组合管理—同时与多个交易所合作时，管理您投资组合的便捷工具
  • Membrana管理基金—使用MBN代币获得保证金合同
